# Spiral Sounds
- This is full stack application which allows a user to find the music records.
- Built using express and sqlite3 for database.
- This is practice project revisit and brush up express fundamentals

## Things covered with this project
- Middleware
- Serving static files
- Creating a database
- Seeding a database
- Retrieving from database
- SQL queries and binding
